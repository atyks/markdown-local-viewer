# Markdown ローカルHTMLビューア (Markdown Local HTML Viewer)

ローカルのMarkdownファイルをブラウザ内だけで読み込み、読みやすく美しくスタイリングされた自己完結型のHTMLへ変換・プレビュー・保存ができるシングルファイルWebアプリケーションです。

---

## 🚀 特徴

1. **完全ローカル動作で安全 (セキュア)**
   - ファイルのアップロードやサーバー通信は一切行いません。ブラウザ上のJavaScriptのみでパース・描画処理を完結させるため、機密ドキュメントも安心して扱えます。
2. **自己完結型HTMLの出力**
   - 変換して保存されるHTMLには、専用のドキュメントテーマ（CSS）がインラインで直接埋め込まれます。CSSファイルや外部フォントを別途用意する必要がなく、単一のHTMLファイルとしてそのまま他者へ共有・配布できます。
3. **豊富なMarkdown記法サポート**
   - **見出し (H1〜H6)**
   - **テキスト装飾** (太字 `**`、斜体 `*`、打ち消し線 `~~`、インラインコード `` ` `` )
   - **リスト** (箇条書き、番号付き、タスクリスト `[x]` / `[ ]` )
   - **テーブル** (列のアライメント `:-:` / `-:` にも対応)
   - **引用 & コールアウト** (GitHub風の `&gt; [!NOTE]`, `&gt; [!TIP]`, `&gt; [!WARNING]` などに対応)
   - **コードブロック** ( ``` を使用した複数行コード記述)
   - **脚注** (`[^1]` とその定義 `[^1]:` )
   - **画像 & リンク** (相対パス、絶対URL、Base64の `data:image/` 形式に対応)
4. **自動目次生成 (TOC)**
   - 「目次を含める」を有効化すると、文書内の見出し2 (H2) および見出し3 (H3) から階層化された目次をタイトル直下に自動生成します。
5. **メタデータ (Front Matter) のパース**
   - Markdownファイルの先頭に記述されたYAML形式のフロントマター (`title`, `subtitle`, `type`, `date`, `updated`, `author` など) を自動で抽出し、読みやすいドキュメントヘッダーに配置します。

---

## 🛠 使い方

1. 本プロジェクトの [markdown-local-viewer.html](file:///app/markdown-local-viewer.html) をブラウザ（Chrome, Firefox, Safari, Edgeなど）で開きます。
2. **「MARKDOWN FILE」** の「ファイルを選択」ボタンをクリックし、手元のMarkdownファイル（`.md` など）を選択します。
   - すぐにプレビューを確認したい場合は、右側の **「サンプルを表示」** ボタンをクリックすることで、内蔵されたサンプルMarkdownの変換結果を確認できます。
3. 必要に応じて、**「目次を含める」** のチェックボックスを切り替えます。
4. 以下のいずれかの方法でドキュメントを活用できます。
   - **別タブで開く**: 変換後のHTMLをブラウザの新しいタブで直接プレビューします。
   - **HTMLとして保存**: 変換後の自己完結型HTMLファイルをローカルにダウンロード・保存します。

## 📝 フロントマターの記述例

Markdownファイルの先頭に以下のような記述（フロントマター）をしておくことで、出力ドキュメントに見栄えの良いヘッダー情報を追加できます。

```markdown
---
title: 週報 - 2026年7月第1週
subtitle: 開発進捗と来週のタスクについて
type: 社内ドキュメント
date: 2026-07-02
author: 開発 太郎
---

# ここから本文
本日の進捗は以下の通りです...
```
